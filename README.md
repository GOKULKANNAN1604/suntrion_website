# Suntrion Website

 A React + TypeScript + Vite website for the Suntrion project.

 ## Overview

 This repository contains the frontend website built with Vite, TypeScript, React, Tailwind CSS,
 and a modern React toolchain.

 ## Getting Started

 1. Install dependencies:
    ```bash
    npm install
    ```

 2. Run the development server:
    ```bash
    npm run dev
    ```

 3. Build for production:
    ```bash
    npm run build
    ```

 4. Preview the production build:
    ```bash
    npm run preview
    ```

 5. Run linting:
    ```bash
    npm run lint
    ```

 ## Project Structure

 - `src/` - application source code
 - `public/` - static assets
 - `index.html` - Vite entry HTML
 - `vite.config.ts` - Vite configuration
 - `tailwind.config.js` - Tailwind CSS configuration
 - `tsconfig.json` - TypeScript configuration

 ## Key Dependencies

 - `react`
 - `react-dom`
 - `react-router-dom`
 - `@vitejs/plugin-react`
 - `tailwindcss`
 - `eslint`

 ## Notes

 This project is configured as a private Vite application and uses modern frontend tooling for
 development and production builds.
